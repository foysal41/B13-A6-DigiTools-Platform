# Project Name + Description

Digitools is digital tools selling platform where user can explore premium digital tools , add them to cart. The website focuses on digital productivity tools like Ai writing tool, design etc. Website are clean and responsive. 
Project URL: https://foysal41-digitools.netlify.app/

## Technologies Used

React.js
Tailwind CSS
DaisyUI
JavaScript (ES6+)
Vite
react-toastify

## Features
1. Dynamic Cart System
- User can add product to cart also can remove product and show total count. This system will real time updating. 

2. Toggle View
- Two button "Product" , "cart". When user clicked product show all product from json and when click cart then show cart items. This time toggle button color will changing 

3. Responsive Design
- Optimized design for mobile, tab, desktop.
